# SD1: Python OOP Type Converter

## Project Overview

This project is a menu-driven Python application that performs a range of type conversions.

The program allows the user to convert:

* Integers to floating-point numbers
* Floating-point numbers to integers
* Integers to strings
* Strings to lists
* Lists to strings
* Integers to binary values
* Integers to hexadecimal values

The application uses a class-based design and includes input validation and error handling.

## Features

## How to Run

From the repository root:

```bash
python SD1/type_converter.py
```

Or from inside the SD1 folder:

```bash
python type_converter.py
```

## Example Usage

```text
--- TYPE CONVERSION MENU ---
1. Integer to Float
2. Float to Integer
3. Integer to String
4. String to List
5. List to String
6. Integer to Binary
7. Integer to Hexadecimal

Choose an option (1-8): 1

Enter an integer: 25

```
