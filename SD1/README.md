# SD1: Python OOP Type Converter

A menu-driven Python program created for my **Fundamentals of Object-Oriented Programming 5N0541** module.

The program uses a `TypeConverter` class with separate methods to convert between integers, floats, strings, lists, binary, and hexadecimal values.

## Skills Demonstrated

- Python classes and methods
- Type conversion
- User input handling
- Loops and conditional logic
- Input validation
- Error handling
- Testing documentation
