# Brief Summary: SD1 Type Conversion Program

## Module

Fundamentals of Object-Oriented Programming 5N0541

## Assignment Title

Skills Demonstration 1: Design, code and test an object-oriented program that performs simple type conversions.

## Task Summary

The task was to create an object-oriented Python program that performs a range of simple type conversions.

The program needed to convert:

- Integers to floating-point numbers
- Floating-point numbers to integers
- Integers to strings
- Strings to lists
- Lists to strings
- Integers to binary
- Integers to hexadecimal strings

## Main Requirements

The assignment required:

- A simple algorithm
- A flowchart
- Pseudocode
- A working Python program
- Object-oriented programming structure
- Appropriate data types
- Logical program blocks
- Clear comments
- Testing and debugging evidence

## Implementation Summary

My solution uses a TypeConverter class with separate methods for each type conversion. The program uses a menu-driven interface so the user can choose which conversion to perform.

The program also includes input validation and error handling to help prevent invalid input from stopping the program.
