# Python Object-Oriented Programming Projects

This repository contains Python projects completed as part of the Fundamentals of Object-Oriented Programming 5N0541 module.

The projects helped me practise object-oriented programming, problem solving, user input validation, error handling, testing, and documentation.

## Projects

### SD1 – Python OOP Type Converter

A menu-driven Python application that converts between different data types including integers, floats, strings, lists, binary values, and hexadecimal values.

Skills demonstrated:

* Object-Oriented Programming (OOP)
* Type conversion
* User input validation
* Error handling
* Loops and conditional logic
* Menu-driven programs

### SD2 – Python OOP ATM Program

A Python ATM simulation that allows users to perform basic banking operations through a menu-driven interface.

Skills demonstrated:

* Object-Oriented Programming (OOP)
* User input handling
* Program design
* Conditional logic
* Data validation

### SD3 – Python OOP ATM Program with Inheritance

An enhanced ATM application that uses inheritance to extend functionality and demonstrate object-oriented design principles.

Skills demonstrated:

* Inheritance
* Classes and objects
* Method reuse
* Program design
* Object-Oriented Programming (OOP)
